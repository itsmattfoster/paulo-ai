🚀 GET STARTED IN 3 STEPS
═══════════════════════════════════════════════════════════════════════════════

STEP 1: INSTALL NODE.JS (if not already installed)
────────────────────────────────────────────────────────────────────────────
   • Go to: https://nodejs.org/
   • Download LTS version
   • Install (keep clicking Next)
   • Restart your computer


STEP 2: INSTALL DEPENDENCIES
────────────────────────────────────────────────────────────────────────────
   Option A (Easy):
      → Double-click: setup.bat

   Option B (Command Line):
      → Open PowerShell in project folder
      → Run: npm install


STEP 3: ADD YOUR OPENAI API KEY
────────────────────────────────────────────────────────────────────────────
   1. Copy .env.example → .env
   2. Get API key from: https://platform.openai.com/api-keys
   3. Edit .env and paste your key:
      OPENAI_API_KEY=sk-your-key-here


🎬 RUNNING THE APP
═══════════════════════════════════════════════════════════════════════════════

   Option A (Easy):
      → Double-click: run.bat

   Option B (Command Line):
      → Open PowerShell in project folder
      → Run: npm run dev

   Then open browser to: http://localhost:3000


📱 HOW TO USE THE APP
═══════════════════════════════════════════════════════════════════════════════

STEP 1: LANDING PAGE
────────────────────────────────────────────────────────────────────────────
   • Enter your website URL (e.g., https://yourwebsite.com)
   • Enter YouTube video URL (must have captions)
   • At least one URL required
   • Click "Analyze & Continue"


STEP 2: ANALYSIS (takes ~1 minute)
────────────────────────────────────────────────────────────────────────────
   The app automatically:
   ✓ Scrapes your website content
   ✓ Extracts YouTube video transcripts
   ✓ Sends to AI for deep analysis
   ✓ Creates your brand context profile
   
   Wait for progress bar to complete...


STEP 3: CREATE CONTENT!
────────────────────────────────────────────────────────────────────────────
   • Select a content mode (Ads, VSL, Emails, etc.)
   • Type what you want to create
   • AI generates content matching YOUR brand voice
   • Continue chatting to refine

⚠️ COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════════════════════

   ISSUE: "npm is not recognized"
   FIX:   Install Node.js from https://nodejs.org and restart computer

   ISSUE: "Failed to analyze YouTube"
   FIX:   Video must have captions/subtitles enabled

   ISSUE: "Failed to analyze website"
   FIX:   Try a different page, some sites block scraping

   ISSUE: "API error"
   FIX:   Check .env file has correct OpenAI API key

   ISSUE: "Port in use"
   FIX:   Close other apps using port 3000

   ISSUE: Dependencies won't install
   FIX:   Delete node_modules folder and run: npm install
